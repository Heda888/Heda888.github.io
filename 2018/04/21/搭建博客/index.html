用hexo+GitHub搭建个人博客

发表于 2018-04-21 | 分类于 教程

从头开始搭建，确实有点难度。但是只要坚持，冷静。多思考思考，就一定能够搭建成功。

在看视频过程中，发现一个问题，就是他们把有的步骤省略了，以致于卡在那里，不知所云。不明白原理，就算是搭建出来了，也不知道如何修改主题，如何写博

客。很是头疼。所以，就从一个小白的角度，以windows操作系统，64位写一个教程，方便大家少走弯路。

这里有我做的思维导图

http://naotu.baidu.com/file/306a4a50b982571003992285c1b2186b大家可以看一下总体框架。

第一步，准备软件

1.node.js(百度搜索进入官网)

选择download，选择相对应的电脑系统和位数

然后安装即可。

2.Get （百度搜素 Git download）


进入下载安装即可。

3.文本编辑器 Sublime Text3

4.markdown2下载安装

5.GitHub账号

第二步，建立本地hexo文件

1.在D盘建立一个命名为My Blog的文件夹（这里一D盘为例，另外的盘和文件名都可）；

2.打开浏览器，输入hexo.io进入hexo官网

3.在My Blog 里面空白处右击，选择“Git Bash here”
将刚才复制的命令npm install hexo-cli -g Paste，回车等待。

4.输入hexo.init
将会下载东西到My Blog文件里面；

5.主题
在https://hexo.io/themes/选择喜欢的主题,进入，找到 Installation,复制里面的git clone https地址,在Git复制，回车;
更新主题：找到My Blog 里面的themes文件位置，右击打开Git ,输入git pull即可。
生成主题：将npm install hexo-generator-json-content –save命令输入到Git；
配置：找到My Blog 目录下的-config.yml，用文本编辑器打开，修改theme,改为刚才下载的主题名字。

6.本地测试
输入hexo s，会出现测试网址，复制到浏览器打开，查看效果。
